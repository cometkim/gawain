# gawain
lightweight cross platform native desktop application framework

NOTE: this project is highly experimental for now. use at your own risk.

## supported platform
- 64-bit windows
- 64-bit macos

##  how to build
```sh
git clone git@github.com:lindenbark/gawain.git
cd gawain
make
```
### prerequisite
- macos
- docker
- nodejs
